The Workshop

This 3 day workshop takes participants through building a full Ember.js application using modern Ember.js patterns available in Ember 2.x like Data down, Actions up, contextual components, closure actions etc.

While no previous Ember.js knowledge is required, participants should be familiar with things like JavaScript, jQuery, HTML, CSS and AJAX as well as the general concept of MVC.

Itinerary

  • Introduction to Ember and Ember CLI
  • The Ember Object Model, Computed Properties and Observers
  • Anatomy of an Ember.js Application
  • Templates and Actions
  • Ember Routing in detail
  • Ember Data and JSON API
  • Components and Component Composition
  • Ember Services
  • Testing - Unit, Integration and Acceptance tests
  • Addons and Dependencies
  • Authentication and Authorization
  • Deployment
  • Real Time Data and Websockets
