The Workshop

This workshop will take participants through building a full Ember.js application complete with a backend API and authentication, all the way to deployment in 3 days. The last day will also cover advanced topics like Animations and realtime data with Websockets.

While no previous Ember.js knowledge is required, you should be comfortable with things like JavaScript, jQuery, HTML, CSS and AJAX. So you can focus 100% on Ember, all assets, CSS and the API server will be provided - all you need is a laptop.

We also offer diversity scholarships for people in underrepresented groups in the tech community.

Lunch at the Workshop Venue is included in the ticket price and there will be a Welcome Party on the first Evening!
