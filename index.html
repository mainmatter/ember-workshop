The Workshop

This workshop will take participants from beginner level to a complete Ember.js application in 3 days. We'll be implementing a complete application, including server API, authentication and deployment. The last day will also cover advanced topics like Animations and realtime data with Websockets.

While no previous Ember.js knowledge is required, you should be comfortable with things like JavaScript, jQuery, HTML, CSS and AJAX in general. To be able to focus on Ember, assets, CSS as well as the API server will be provided by us - you only need to bring a laptop.

Lunch at the Workshop location is included in the ticket price and there will also be a Welcome Party on the first Evening!
