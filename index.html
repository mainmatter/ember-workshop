The Workshops

We organize workshops that teach participants how to build ambitious Ember.js applications using modern and idiomatic Ember.js patterns.

Our workshops range from beginner trainings in which we take participants through building a full Ember.js application to advanced workshops on techniques like composition of complex component hierarchies and Data Down, Actions Up, FastBoot, Authentication and Authorization etc.

3 day Ember.js Jumpstart

This 3 day workshop takes participants through building a full Ember.js application using modern and idiomatic Ember.js patterns like Data down, Actions up, contextual components, closure actions etc.

While no previous Ember.js knowledge is required, participants should be familiar with things like JavaScript, jQuery, HTML, CSS and AJAX as well as the general concept of MVC.

Workshop Format
  • Topic are introduced via in-depth presentations.
  • Small demo applications are used to illustrate how different Ember.js concepts are used in practice.
  • Participants will actively make use of the concepts covered in each block in mentored coding sessions.
  • All reference materials incl. presentation slides, demo applications and reference solutions for the coding session are handed out to participants.
Time Schedule
Day 1
  • Introduction to Ember's core concepts
  • Ember CLI
  • Templates and Actions
  • Ember Routing in detail
Day 2
  • Ember Data and JSON API
  • Components and Data Down, Actions Up
  • Services
Day 3
  • Testing
  • Addons and Dependencies
  • Authentication and Authorization
  • Deployment
