The Workshop

This 3 day workshop takes participants through building a full Ember.js application using modern and idiomatic Ember.js patterns like Data down, Actions up, contextual components, closure actions etc.

While no previous Ember.js knowledge is required, participants should be familiar with things like JavaScript, jQuery, HTML, CSS and AJAX as well as the general concept of MVC.

Itinerary

  1. Introduction to Ember and Ember CLI
  2. The Ember Object Model, Computed Properties and Observers
  3. Anatomy of an Ember.js Application
  4. Templates and Actions
  5. Ember Routing in detail
  6. Ember Data and JSON API
  7. Components and Component Composition
  8. Ember Services
  9. Testing - Unit, Integration and Acceptance tests
  10. Addons and Dependencies
  11. Authentication and Authorization
  12. Deployment
  13. Real Time Data and Websockets
