3 day Ember.js Jumpstart

This 3 day on-location workshop takes participants through building a full Ember.js application using modern and idiomatic Ember.js patterns like Data down/Actions up, contextual components, closure actions etc. We'll introduce all of Ember.js' concepts required to create ambitious, production ready applications in a step by step manner.

While no previous Ember.js knowledge is required, participants should be familiar with JavaScript, jQuery, HTML, CSS and AJAX as well as the general concept of MVC.

Workshop Format
  • Each topic is introduced via an in-depth presentation.
  • Small demo applications are used to illustrate how the different Ember.js concepts are used in practice.
  • Participants will actively make use of the concepts covered in each block in mentored coding sessions - step by step building a complete application in the process.
  • All reference materials incl. presentation slides, demo applications and reference solutions for the coding session are handed out to participants.
Time Schedule
Day 1

  • Ember CLI

Day 2
  • Ember Data and JSON API
  • Components and Data Down/Actions Up
  • Services
Day 3
  • Testing
  • Addons and Dependencies
  • Authentication and Authorization
  • Deployment

Advanced Session: Authentication and Authorization

This half-day online workshop is a deep dive into authentication and authorization from the creators of Ember Simple Auth. We will cover fundamental auth concepts, discuss different authentication mechanisms like username/password and OAuth 2.0 and discuss related security concerns.
